# My Personal Profile & Business Demo

This project is a collection of static HTML pages created as part of the **JSD-12 (Junior Software Developer)** course, Week 03. It serves as a demonstration of basic web development skills, including semantic HTML, inline CSS, forms, tables, and media embedding.

## Project Overview

- **Purpose:** Educational/Classroom Exercise.
- **Main Technologies:** HTML5, CSS (Inline), Media (Video, Images).
- **Theme:** Personal branding and a fictional "Boom Boom Hammer" business.

## Key Files

- **`index-personal.html`**: The main personal profile page for Pathomrat Anusonpornpherm (Max), featuring personal interests, skills, and a video bio.
- **`index-Business.html`**: A product showcase for "Boom Boom Hammer" (Thor's Hammer), detailing its "divine" features and ordering steps.
- **`business-form.html`**: A dedicated order form for the "Boom Boom Hammer" business, including product selection and contact details.
- **`form.html`**: A comprehensive practice file for various HTML form elements (radio buttons, checkboxes, number inputs, etc.).
- **`table.html`**: A demonstration of HTML tables, used here to display a "Song Profile".
- **`81j3g7wlnIL...jpg`**: Profile image for the personal page.
- **`thors-hammer...avif`**: Product image for the business page.
- **`InWhite Ep.4...mp4`**: Background/Interest video for the personal profile.

## Running the Project

As this is a static project, no build or installation steps are required:

1.  Open any of the `.html` files (e.g., `index-personal.html`) directly in your preferred web browser.
2.  Navigate between pages using links (where provided, e.g., from the business page to the order form).

## Development Conventions

- **Styling**: Styles are applied primarily using inline `style` attributes on HTML elements.
- **Assets**: Images and videos are stored in the root directory and referenced with relative paths.
- **Language**: Content is provided in a mix of English and Thai.
